# Yihu Hou Academic Website

This repository contains a lightweight single-page academic website for GitHub Pages.

## Main Files

- `index.html`: page content
- `assets/styles.css`: layout and styling
- `assets/script.js`: small interaction effects
- `assets/yihu-photo.jpg`: web-optimized profile photo

## Current Structure

The site currently includes:

1. Intro section with photo and current appointment
2. Research section with three projects
3. HTML CV section with appointment, education, and teaching
4. Contact section

## Publish

1. Commit and push to `https://github.com/yihuhou/yihuhou.github.io`
2. In GitHub, open `Settings` -> `Pages`
3. Set the source to the default branch and `/ (root)`
4. Wait for deployment to finish
